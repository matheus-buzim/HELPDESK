📩 Sistema de Chamados via E-mail

Este projeto é um estudo em PHP desenvolvido a partir de uma necessidade real identificada durante minha atuação como estagiário de suporte no IPAMV.

Na época, os chamados eram realizados via WhatsApp e e-mail, sem qualquer tipo de controle estruturado. Isso impossibilitava o registro adequado das demandas, além de não permitir a extração de indicadores como volume de chamados, principais causas, sistemas mais afetados e SLA de atendimento.

Com base nesse cenário, o projeto tem como objetivo criar um sistema de chamados integrado ao e-mail, permitindo organizar, registrar e acompanhar as solicitações de forma centralizada.

🎯 Objetivos
- Centralizar os chamados recebidos por e-mail
- Criar histórico de atendimentos
- Permitir análise de dados (causas recorrentes, SLA, volume)
- Melhorar a organização e rastreabilidade das demandas

🛠 Tecnologias
- PHP
- XAMPP (MySQL) como banco de dados
